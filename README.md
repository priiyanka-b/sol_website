# SOL — Premium Homepage Redesign

A production-ready B2B pre-roll cone manufacturing and custom packaging website concept for SOL.

## Design Features

- **Clean light theme** — Minimalist design with white, soft green, and charcoal tones
- **Premium aesthetic** — Inspired by Apple, Stripe, Linear, Notion, and Webflow Enterprise
- **Scroll animations** — Framer Motion reveal effects on every section
- **Mobile-first** — Fully responsive across all breakpoints
- **Conversion-focused** — Multiple CTAs, instant quote pricing estimator, strategy call scheduling

## Homepage Sections

1. Hero with floating product visuals and statistics
2. Trusted By leading cannabis brands
3. Product Categories grid (6 categories)
4. Why Choose SOL
5. Manufacturing Process timeline
6. Featured Products showcase
7. Testimonials with trust indicators
8. Industry Resources / blog cards
9. Strong CTA banner
10. Modern footer

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React icons
